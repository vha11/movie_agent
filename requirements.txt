# NOTA IMPORTANTE — ¿Por qué no scrapeamos IMDb directamente?
# IMDb pertenece a Amazon y no ofrece API pública gratuita estándar.
# El acceso a sus datos es de pago o restringido (AWS datasets, licencias comerciales).
# Además, IMDb bloquea activamente los scrapers mediante:
#   - Cloudflare anti-bot
#   - Rate limiting agresivo
#   - Cambios frecuentes de estructura HTML
# Solución: usamos OMDb (Open Movie Database), la alternativa oficial y gratuita.
# OMDb expone los mismos datos de IMDb (ratings, votos, sinopsis, director...)
# a través de una API JSON estable. Gratuita hasta 1.000 peticiones/día.
# Registro en: https://www.omdbapi.com/apikey.aspx
